Potential App Idea Brainstorming

  • Live Listening Translator

    This would be a web application where you give it access to your microphone and it visualizes any audio it hears into a different language It would be very useful for someone either working with a different language or simply trying to learn something new. Easily can run in the background on a monitor or TV.

  • Language Translate Popularity

    A web application which shows which languages are most often requested to translate from and to. Specifically, which words are most often translated in the specified language. More of a fun application than a useful one, though could be helpful in any type of related research.

  • Translation Conversationalist

    A live chat bot gone abroad. Ask a question through the chat and the bot will respond in your requested language. This will utilize both an API for responses and translation. Helpful for anyone looking to improve their conversation skills in a different language.
